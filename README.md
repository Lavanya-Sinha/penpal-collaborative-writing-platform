Title - PenPal – Collaborative Writing Platform

A full-stack collaborative writing platform that allows users to create, edit, manage, and collaboratively work on stories using a rich text editor. The platform supports JWT-based authentication, story collaboration, comments, and user-specific content management.

Features:
- JWT authentication
- Collaborative story editing
- Rich text editor using React-Quill
- Story CRUD functionality
- Story-specific collaborators
- User profile and story management
- Comments and reader interaction
- Redux-based global state management
- Protected routes and authenticated APIs

Tech-Stack:
Frontend:
- React.js
- Redux
- Fluent-ui
- React-Quill
Backend:
- Node.js
- Express.js
- JWT
Database:
- MySQL
